# Zac v5 画面一覧・画面遷移図

作成日: 2026-04-27  
基準文書: `Zac_planning_v5.md`

---

## 1. 目的

Zac MVPの画面構成とユーザー導線を固定し、実装時の抜け漏れを防ぐ。

---

## 3. 主要画面遷移

### 3.1 初回登録

```text
AUTH-01
  -> AUTH-03
  -> ONB-01
  -> ONB-02
  -> ONB-03
  -> ONB-04
  -> ONB-05
  -> ONB-06
  -> HOME-01
```

### 3.2 ジムから予定作成

```text
HOME-01 or EXP-01
  -> GYM-01
  -> GYM-02
  -> PLAN-03
  -> PLAN-02
```

### 3.3 予定から記録

```text
PLAN-01
  -> PLAN-02
  -> PLAN-06
  -> LOG-03
  -> LOG-05
```

### 3.4 記録から投稿

```text
LOG-05
  -> POST-02
  -> POST-01
```

### 3.5 通報

```text
POST-01 or PLAN-02 or LOG-02 or USER-01
  -> SAFE-01
  -> 元画面へ戻る
```

## 4. 状態別画面

| 状態 | 対象画面 | 表示 |
|---|---|---|
| 未ログイン | ホーム/探す | ジム、イベントは閲覧可。保存、参加、作成、コメント時にログイン誘導 |
| 未ログイン | 予定/記録/投稿/マイページ | タブには表示せず、直URLではログイン誘導 |
| 空データ | 予定一覧 | 予定作成CTA |
| 空データ | 記録一覧 | 記録作成CTA |
| 空データ | 保存ジム | 探すCTA |
| 読み込み中 | 一覧/詳細 | skeletonまたはspinner |
| 通信失敗 | 全API画面 | 再試行CTA |
| 権限なし | 詳細画面 | 表示できない旨 |
| 削除済み | 詳細画面 | 存在しない扱い |

## 5. 権限別表示

| 対象 | 本人 | 他ユーザー | 管理者 |
|---|---|---|---|
| 自分の投稿 | 編集/削除可 | 表示範囲内で閲覧 | 非表示/削除可 |
| 自分の記録 | 編集/削除可 | 表示範囲内で閲覧 | 原則閲覧制限。通報時のみ |
| 自分の予定 | 編集/キャンセル可 | 参加/コメント可 | 非表示/削除相当可 |
| ジム | 保存/予定作成 | 保存/予定作成 | 編集可 |
